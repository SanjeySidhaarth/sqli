About This Application

The SQL Injection Detection and Prevention System is designed to protect web applications from malicious database attacks.

This system uses Machine Learning models, primarily Random Forest, to analyze user inputs and detect SQL injection patterns in real time.

Repeated malicious attempts automatically trigger email-based verification and Google authentication to ensure secure access.
